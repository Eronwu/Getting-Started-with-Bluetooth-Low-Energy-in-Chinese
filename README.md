## 关于这个项目

这是一本 2014 年出版的 BLE 入门经典。原书基于蓝牙 4.0/4.1 规范，虽然蓝牙技术如今已迭代到 6.0 版本，但书中对 BLE 基础概念（GATT、GAP、Bonding、Encryption 等）的讲解依然清晰易懂，是很好的入门读物。

本书最初于2020年由 [Eron Wu](https://github.com/Eronwu) 出于兴趣翻译完成。2026 年，借助**大语言模型**的能力，对全书进行了逐章对照原书原文的全面修订——修正了初版中的误译，统一了专业术语，重写了大量生硬的直译句子，使译文更加通顺准确。

此外，考虑到原书出版至今已有十余年，蓝牙技术发生了翻天覆地的变化，本次中文翻译版本主动增加了两章增订内容：**第十一章**梳理了 BLE 从 4.0 到 6.0 的技术演进，**第十二章**介绍了现代 BLE 开发所需的工具链和最佳实践。希望这些内容能帮助读者在理解基础概念的同时，也能了解当前的技术现状。

## 译者注

专业名词翻译尽量根据行业习惯取舍：

- "蓝牙低功耗" 全书多用 BLE 缩写
- Host → "主机"，Controller → "控制器"
- 可能带来歧义或技术交流中更常用英文的词，保留中英文对照，如时序（timing）、连接（connections）
- 公司名称保留原文，如 Nordic Semiconductor
