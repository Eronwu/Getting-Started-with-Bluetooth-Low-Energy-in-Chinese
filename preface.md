# 前言

低功耗蓝牙（BLE）作为蓝牙 4.0 核心规范的一部分推出，是一项出色的无线技术，使手机应用开发者能够利用主流移动操作系统，通过简单可靠的连接与大量外部硬件设备进行数据交互。

本书旨在提供对 BLE 可靠、实用且高层次的理解：数据如何组织、设备如何相互通信、关键设计决策以及与其他无线技术的权衡。这将使你对 BLE 有足够的了解，从而更有信心地使用当前大多数嵌入式设备和移动操作系统提供的高级 API 进行开发，并在你进一步深入研究时，能够理清那些深奥的技术文献中的专业术语和命名惯例。本书还将阐明 BLE 相对于其他无线技术（如 WiFi、NFC、经典蓝牙、Zigbee 等）的优势和局限性。

经验丰富的嵌入式固件工程师将能够更好地进入现有技术文献进行深入研究，手机应用开发者将更清晰地了解数据如何在 BLE 设备中组织并与现有硬件交互。

## 这本书给谁

此书面向两类读者：

*手机应用开发者*

  首先，本书面向那些希望设计与外部物理设备通信的应用的手机应用开发者。面对官方 2600 页的蓝牙 4.1 核心规范，本书提供了一个关于 BLE 高级概念的清晰综述，是更好的入门途径。

*嵌入式工程师*

  另一方面，本书也面向考虑在产品设计中使用 BLE 的传统嵌入式工程师。如果你需要快速了解 BLE 能做什么，本书帮助你快速评估它作为无线协议在你项目中的优缺点。

## 如何使用这本书

这本书分为三个主要部分。

### BLE 概述

前四章提供了 BLE 作为技术的高级综述，解释了数据如何组织以及关键限制，同时也介绍了你在使用 BLE 过程中会遇到的所有关键概念：

*第一章 介绍*

  第一章介绍了 BLE 作为无线标准的基本概念。简要描述了了解这项技术最重要的元素所需的前提条件，并提供了当前可用的不同规范和芯片配置的概览。本章还介绍并解释了 BLE 的基本概念，如广播（broadcasting）、连接（connections）以及设备可以扮演的各种角色。

*第二章 协议基础*

  本章专注于 BLE 的完整实体：协议栈。这里概括了每个协议层及其关键特性，从规范中过滤掉与 BLE 应用开发者没有直接联系的细节。描述了每一层在整个协议栈中的角色，并特别注意这些角色在真实场景中的影响。

*第三章 GAP（广播和连接）*

  本章介绍 GAP（Generic Access Profile，通用访问配置文件），负责管理广播和连接。概述了允许多个设备同时使用广播数据包传播信息和通过连接交换数据进行交互的模型和流程。

*第四章 GATT（服务和特征）*

  本章提供 GATT（Generic Attribute Profile，通用属性配置文件）的概述，GATT 建立了 BLE 内用于表示和操作数据的层次结构和格式。介绍了服务和特征的基本概念，以及允许设备相互交互数据的流程。

### 开发测试工具

接下来三章展示了一些实用工具（包括软件和硬件），用于开发或测试具有 BLE 功能的应用和设备。这些章节专注于便宜、易用的工具，帮助你快速开始研发，而无需投入数千美元：

*第五章 硬件平台*

  本章为产品设计师提供了一些前沿的带 BLE 外围设备或产品的嵌入式开发平台的概述。

*第六章 调试工具*

  无论你是在设计自己的硬件，还是在与现有硬件通信的应用，你几乎肯定需要花很多时间进行调试。调试无线设备的过程与纯软件环境下的开发不同。本章展示了一些用于 BLE 工作的实用调试工具，查看空中传输的实际内容。

*第七章 应用设计工具*

  本章展示了一些为 BLE 手机应用开发者准备的关键工具。这些工具将帮助你快速测试、验证你的软件，甚至在早期设计流程中没有真实硬件时模拟设备环境。

### 开发平台

最后三章介绍了一些用于 BLE 相关开发的主流开发平台（应用开发者使用的 iOS 和 Android，以及用于产品设计和嵌入式硬件开发者的各类嵌入式平台）：

*第八章 Android 编程*

  本章提供了在 Android 操作系统上实现 BLE 功能所需的硬件、软件和流程的基本概述。

*第九章 iOS 编程*

  本章探索了 iOS 7 上支持 BLE 应用开发的框架、类和方法的关键内容。展示了一个使用 BLE 读取外围设备电池电量信息的应用开发示例，以及使用 iBeacon 进行定位的示例。

*第十章 嵌入式应用开发*

  本章介绍了嵌入式设备编译代码所需的工具。使用第五章介绍的 ARM 架构下免费开源的 GNU 工具链和跨平台编译，以及第五章介绍的 nRF51822-EK 平台，你将创建并开发一个心率监测仪演示程序。
